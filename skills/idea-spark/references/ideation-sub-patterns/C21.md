# C21
_parent: **Reframe as a Solvable Object** (`reframe_as_solvable_object`)_

## Tactical pattern

This cluster recasts reinforcement-learning and sequential-control problems — which natively require reward signals, action labels, or expert demonstrations — as supervised or generative-sequence-modeling problems by manufacturing the absent supervision from interaction data that already exists. The concrete moves are: retrospectively relabel each collected trajectory's target to the outcome it actually achieved (turning suboptimal traces into optimal demonstrations for pure distribution matching); discretize continuous observations or actions into a token vocabulary and forecast them autoregressively as a sequence; and pre-train a frozen generative world model or behavioral prior on cheap unlabeled/task-agnostic interaction, then route downstream control through it. When labels are entirely absent, the missing reward/action component is itself replaced by a self-supervised proxy — future reachability, multi-step inverse dynamics, latent behavioral modes, or a low-capacity causal bottleneck that recovers a latent action space from consecutive states. The payoff is that mature supervised/generative machinery (transformers, normalizing flows, VQ encoders, maximum-likelihood estimation) applies directly, eliminating online optimization, reward design, or the need for gold-standard demonstrations.

## Step-by-Step

1. Identify which supervision component the native formulation silently assumes but cannot obtain — optimal demonstrations, dense reward labels, or ground-truth action annotations — and reframe the bottleneck as a missing-supervision problem rather than an architecture or loss-function problem. The whole tactic hinges on locating the absent ingredient precisely, because the recast will be built around fabricating exactly that ingredient.
2. Choose a target object class whose mature solvers do NOT need the missing component: supervised likelihood / distribution matching, autoregressive sequence prediction, or an invertible generative prior. Prefer an object whose machinery is self-contained and battle-tested in adjacent fields, so the contribution is the reframing plus the substitution, not the invention of a new solver.
3. Manufacture the missing supervision from data that already exists: retrospectively relabel each achieved outcome as its own optimal target, derive a self-supervised proxy signal (future reachability, inverse dynamics, latent mode), or pre-train a frozen generative prior/world model on cheap unlabeled or task-agnostic interaction. Make the substituted signal load-bearing and faithful — avoid exact-algebraic equivalences that hold only under brittle assumptions, since a substitution that is only approximately valid forfeits the guarantees the recast was meant to import.
4. Make the recast tractable by compressing the continuous, high-dimensional native space into the object's natural representation — tokenize observations/actions into a discrete vocabulary, quantize the latent goal space into a prototype set — and verify the compression preserves the modes and diversity the native problem requires. Collapse to a single behavioral mode (unimodal imitation) is the diagnostic that the compression destroyed what the recast needed to capture.
5. Demonstrate that the recast unlocks a capability the native formulation lacked, at a scale that pre-empts the 'analysis-only' and 'toy-only' critiques: post a clear empirical win over native-formulation baselines on non-trivial benchmarks, ablate that the gain comes from the manufactured-supervision move rather than incidental architecture, and explain any boundary behavior rather than leaving an unexplained degradation that undercuts the generalization claim.

## Differentiation within parent

The solvable object this cluster reaches for is supervised learning or autoregressive generative modeling, and its signature sub-move is fabricating the supervision the native problem lacks — relabeling achieved outcomes as targets, or substituting reward/action labels with self-supervised proxies. Contrast cluster 27, which recasts the problem as a differentiable multi-agent game whose Nash equilibrium coincides with the desired ranked decomposition: that cluster engineers competing local objectives and reads off an equilibrium, never manufacturing missing labels and never invoking generative/supervised solvers. Contrast also cluster 0, which keeps the decision-scoring frame intact and enriches it with interpretable cumulative sub-objective predictors embedded directly in the scoring computation, rather than wholesale converting the control problem into generative-sequence training. Cluster 21 is the one where 'no rewards / no action labels / no demos' is converted into 'self-generated labels,' and the recast target is a generative or supervised learner that existing machinery can solve off the shelf.

## When to pick this one

Pick this instantiation when the sequential-decision problem is blocked specifically by absent supervision — no reward signal, no action annotations, or no optimal demonstrations — yet abundant unlabeled or task-agnostic interaction data is available to manufacture targets from. If instead the blocker is competitive or allocation structure (favoring a game/equilibrium recast) or an opaque scoring computation that needs interpretable predictors, a different sibling fits better.

## Tactical failure mode

Papers attempting this recast fail when the reframe does not actually unlock a capability the native formulation lacked at a credible scale. One recurring shape: the recast is deployed as a descriptive evaluation or analysis lens — for instance relabeling objectives post-hoc to score collected data — but the accompanying method posts no clear empirical win, so reviewers read it as a benchmark study rather than a contribution. A second shape: the substitution of the missing component rests on a brittle exact assumption, such as the target objective being an exact algebraic combination of source objectives, whose solving machinery does not scale and whose formal guarantees evaporate the moment the assumption holds only approximately. A third shape: the recast is validated only on toy environments and an unexplained boundary behavior — performance degrading past a certain horizon or context length — directly undercuts the generalization claim the reframe was supposed to deliver. Self-check: can you show the manufactured-supervision object beating native-formulation baselines on a non-trivial benchmark, with the gain attributed to the recast and any boundary behavior explained, rather than offering observations or toy-scale wins?

## Examples
### Oral lessons
- Retrospectively reassigning each trajectory's target to the outcome it actually reached turns every suboptimal trace into a valid optimal demonstration, collapsing a two-stage estimate-then-improve loop into a single supervised distribution-matching fit.
- Discretizing a continuous output space into a token vocabulary lets a standard autoregressive sequence model represent multi-modal behavior that continuous regression would collapse into a single averaged mode.
- The pre-trained artifact should capture the input-to-output (action) distribution, not just input representations, because in sequential decision problems it is the output space — not the observation space — that governs downstream difficulty.
- An invertible, bijective generative prior can reparameterize the action space so that every original action stays reachable, biasing search toward useful outputs without amputating coverage.
- Absent reward or action labels can be replaced by self-supervised proxies — future reachability, multi-step inverse dynamics, or latent behavioral modes — that still preserve valid recursive/bootstrapped updates, so unlabeled sequences become decision-quality training data.
- A low-capacity causal bottleneck trained jointly with a next-state predictor forces all steering information through a recovered latent control variable, reconstructing an action space from purely unlabeled interaction.

### Reject lessons
- A reframe used only as an evaluation or analysis lens, without a method that posts a clear win, reads as a descriptive benchmark and falls below the novelty bar even when the framing is reusable.
- A recast resting on an exact algebraic equivalence (target = exact polynomial of sources) inherits brittle, non-scaling machinery, and its formal guarantees collapse the moment the equivalence is only approximate.
- Validating the recast only on toy environments, with an unexplained boundary failure such as performance degrading past a certain context or horizon length, directly undercuts the generalization claim the reframe was meant to establish.
